Elementos de Cabeçalho

Voltar

Os elementos HTML <h1> ao <h6> representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo. Um elemento de cabeçalho descreve brevemente o tópico da seção em que ele está. As informações de cabeçalho podem ser usadas por agentes de usuário, por exemplo, para construir uma tabela de conteúdos para um documento automaticamente.

<p>

Voltar

O elemento HTML <p> representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

<mark>

Voltar

O elemento HTML <p> representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

<small>

Voltar

O elemento HTML representa comentários laterais e letras pequenas, como direitos autorais e texto legal, independentemente de sua apresentação estilizada. Por padrão, ele renderiza o texto dentro dele com um tamanho de fonte menor.

<i>

Voltar

O elemento HTML representa uma faixa de texto que se destaca do texto normal por algum motivo, como texto idiomático, termos técnicos, designações taxonômicas, entre outros. Historicamente, estes têm sido apresentados em itálico, que é a fonte original da nomenclatura deste elemento.

<u>

Voltar

O elemento HTML representa um trecho de texto embutido que deve ser renderizado de forma a indicar que possui uma anotação não textual. Isso é renderizado por padrão como um sublinhado sólido simples, mas pode ser alterado usando CSS.

<strong>

Voltar

O elemento HTML indica que seu conteúdo tem forte importância, seriedade ou urgência. Os navegadores normalmente renderizam o conteúdo em negrito.

<ol>

Voltar

O elemento HTML representa uma lista ordenada de itens — normalmente renderizada como uma lista numerada.

<ul>

Voltar

O elemento HTML representa uma lista não ordenada de itens, normalmente renderizada como uma lista com marcadores.

<li>

Voltar

O elemento HTML é usado para representar um item em uma lista. Deve estar contido em um elemento pai: uma lista ordenada ( ), uma lista não ordenada ( ) ou um menu ( ). Em menus e listas não ordenadas, os itens da lista geralmente são exibidos usando marcadores. Em listas ordenadas, eles geralmente são exibidos com um contador crescente à esquerda, como um número ou uma letra.

<a>

Voltar

O elemento HTML (ou elemento âncora ), com seu atributo , cria um hiperlink para páginas da Web, arquivos, endereços de e-mail, locais na mesma página ou qualquer outra coisa que um URL possa endereçar. O conteúdo de cada um <a> deve indicar o destino do link. Se o atributo href estiver presente, pressionar a tecla Enter com o foco no elemento <a> irá ativá-lo.

<hr>

Voltar

O elemento HTML representa uma quebra temática entre elementos de nível de parágrafo: por exemplo, uma mudança de cena em uma história ou uma mudança de tópico dentro de uma seção.

<sub>

Voltar

O elemento HTML especifica o texto embutido que deve ser exibido como subscrito apenas por motivos tipográficos. Subscritos são normalmente renderizados com uma linha de base reduzida usando texto menor

<sup>

Voltar

O elemento HTML especifica o texto embutido que deve ser exibido como sobrescrito apenas por motivos tipográficos. Os sobrescritos geralmente são renderizados com uma linha de base elevada usando um texto menor.

<blockquote>

Voltar

O elemento HTML indica que o texto incluído é uma citação estendida. Normalmente, isso é renderizado visualmente por recuo (consulte as Notas para saber como alterá-lo). Uma URL para a fonte da citação pode ser fornecida usando o atributo, enquanto uma representação de texto da fonte pode ser fornecida usando o elemento.

<font>

Voltar

O elemento HTML define o tamanho da fonte, a cor e a face de seu conteúdo.

Obsoleto: esse recurso não é mais recomendado. Embora alguns navegadores ainda possam suportá-lo, ele pode já ter sido removido dos padrões da Web relevantes, pode estar em processo de eliminação ou pode ser mantido apenas para fins de compatibilidade.

<del>

Voltar

O elemento HTML representa um intervalo de texto que foi excluído de um documento. Isso pode ser usado ao renderizar "controlar alterações" ou informações de diferenças de código-fonte, por exemplo. O elemento pode ser usado para o propósito oposto: para indicar o texto que foi adicionado ao documento.

<abbr>

Voltar

O elemento HTML representa uma abreviação ou acrônimo. Ao incluir uma abreviação ou acrônimo, forneça uma expansão completa do termo em texto simples no primeiro uso, junto com a marcação da abreviação. Isso informa ao usuário o que significa a abreviação ou acrônimo.
